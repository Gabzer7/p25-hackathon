# le taquin (aka puzzle 8)

## le sujet

le sujet du taquin de Thierry est ici:

https://flotpython-exos-python.readthedocs.io/en/devel/tps/puzzle8/README-puzzle8-nb.html

## organisation du travail en équipe

pour ce hackathon vous pouvez sans souci viser les deux objectifs décrits dans
le TP, à savoir le solver et l'interface graphique

vous pouvez répartir les rôles en gros:

- une personne s'occupe du solver
- deux personnes s'occupent de l'interface graphique
- un chef de projet qui coordonne les efforts et contribue éventuellement dans les deux domaines

mais bien sûr c'est juste une suggestion pour vous donner une idée du volume de
travail  
le solver est un ardu mais demande peu de code en fin de compte
